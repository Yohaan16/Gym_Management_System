Place Lottie .json animations here (e.g., bicep_curl.json, squat.json, push_up.json).

Recommended files:
- bicep_curl.json
- push_up.json
- squat.json
- lunge.json
- plank.json
- default.json (fallback)

You can download free animations from:
- https://lottiefiles.com
- https://iconscout.com/lottie
- https://lordicon.com

After adding files, run:
  flutter pub get
and rebuild the app.
